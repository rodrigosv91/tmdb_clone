# TMDB CLONE

## 1. Pré-requisitos

Node.js: Certifique-se de que o Node.js (versão 14 ou superior) está instalado em sua máquina. Você pode baixar a versão mais recente [aqui](https://nodejs.org/).

Baixe o instalador do Git para Windows [aqui](https://git-scm.com/download/win) e execute o arquivo baixado. Siga as instruções na tela para completar a instalação.

## 2. Clonando o Repositório

Abra o terminal e execute o seguinte comando para clonar o repositório:

```bash
git clone https://github.com/rodrigosv91/tmdb_clone.git
```

## 3. Instalando Dependências

```bash
cd tmdb_clone
npm install
```

## 4. Configuração das Variáveis de Ambiente

No arquivo .env na raiz do projeto substitua "SUA_CHAVE_API" pela sua chave da API do TMDB e salve:

## 5. Executando o Projeto Localmente

```bash
npm start
```
